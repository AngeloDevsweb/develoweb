Desarrollo Web

Diseño Responsivo

Es muy importante que tu pagina tenga un diseño responsivo el cual le permita poder verse de forma estetica en cualquier dispositivo, un smartphone, tablet o pc tienen que poder visualizar de manera agradable el contenido que le estas presentando al usuario.

Tecnologias Web

Existen muchas tecnologias con las cuales puedes desarrollar distintos tipos de aplicacion al igual que paginas web, lo ideal es que comienzes con una tecnologia en especifico y la uses hasta tener un dominio sobre ella, luego te daras cuenta que otros lenguajes y herramientas son similiares y te será muy facil.

Posicionamiento SEO

Existen reglas fundamentales para mejorar el Posicionamiento de tu pagina web o blog de articulos, debes tener en cuenta estas reglas para tener una mejor posicion en google o en el buscador de tu preferencia.

Servidor o Hosting compartido

Para poder publicar tu pagina web necesitaras un servidor donde se aloje tus archivos estaticos y puedas enlazar tu pagina con un nombre de dominio, y así poder encontrarla en un buscador ya sea google u otro.

Frameworks para Frontend de javascript

Vue Js

Vue es un framework Open Source de javascript, el cual nos permite construir interfaces de usuario de una forma muy sencilla, el aprendizaje a mi parecer no es tan complejo pero todo depende claro de las bases que tengas en javascript y el manejo callbacks, promesas, funciones, entre otros temas más y sobre todo objetos.

React Js

React es una libreria de javascript la cual tiene como objetivo crear interfaces de usuario, sin embargo la verdad es que con react podemos diferentes tipos de aplicaciones web como la SPA (Single Page Application) o incluso aplicaciones moviles con react native que esta basada en React js.

Angular

Es un framework opensource desarrollado por Google para crear aplicaciones web SPA (Single Page Application). Angular separa el frontend del backend y maneja con orden los archivos ya que utiliza MVC (modelo-vista-controlador) todo eso posibilita el poder realizar cambios y modificaciones de manera sencilla en tu web.

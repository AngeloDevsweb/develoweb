ANGULAR

Angular es uno de los frameworks más populares para frontend a dia de hoy, y eso se basa en que Angular es bastante robusto, con una cantidad muy extensa de bibliotecas que permiten poder estructurar las aplicaciones de manera correcta.

Es un framework basado en javascript pero que usa Typescript para su codificación ya que este ofrece otras opciones sobre js. La potencia que tiene este frame es muy buena de hecho llega a tener cierta complejidad al momento de programar con él.

Con angular puedes desarrollar una web SPA (single page application) y crear las interfaces de usuario para poder ofrecer unas buenas vistas al cliente, pero debes tener en cuenta que angular no ofrece la libertad que pueden tener otros framework o librerías como ser el caso de React o Vue, ya que angular tiene una serie de normas y reglas que debes seguir para que tu aplicación funcione correctamente.

Ademas de servir para crear paginas SPA, también nos da la posibilidad de trabajar con backend, usando la implementación del Server Side Rendering. A esta implementación se le llama Angular Universal.

¿Cuándo puedo usar angular?

Angular es un framework completo, robusto que te permitirá un buen desarrollo, pero tienes que saber en qué momento vas a usar angular. Por todas las características que ofrece angular sería muy bien utilizado para proyectos medianos o grandes, que necesiten una buena calidad de código, mantenible y sobre todo escalable.

Este framework nos ofrecerá muchas herramientas no solo para SPA sino también para crear las PWA (progresive web apps) que son aplicaciones web que funcionan de forma similar a una aplicación móvil.

Tu tendrás que tomar la decisión de que framework utilizar al momento de implementar tu página web o app, pero si tu aplicación será algo pequeña o no tan robusta a largo plazo entonces podrias tomar una opción como Vue o React, pero si tienes un proyecto grande sería muy bueno que aprendieras a implementar Angular.

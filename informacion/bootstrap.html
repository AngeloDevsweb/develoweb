¿Cómo usar Bootstrap en tu proyecto de React js?

Si ya instalaste react js seguramente ahora quieras empezar a construir tu primera pagina web, y para ello necesitas instalar bibliotecas que te permitan poder montar tu web de forma eficiente.

Una parte muy importante de una pagina web son los estilos, el formato que le vayas a dar a tu web es aquello que va a llamar la atención por eso tienes que dedicarle una buena parte de tu tiempo, pero si el caso fuera que necesitas ahorrar un poco de tiempo y no dispones de mucho para empezar a montar una web entonces tal vez necesites usar un framework de css, para que así no tengas que dedicar mucho tiempo en crear los estilos desde 0.

Y en esta guía te voy a mostrar como implementar bootstrap en tu proyecto.

Lo primero que debes hacer es ir a la documentación de bootstrap de allí vas a copiar el comando para poder instalar bootstrap en tu proyecto, también podrías utilizar la cdn que te ofrece bootstrap pero no seria lo mas recomendado ya que vas a estar trabajando con ficheros de javascript, entonces lo mejor que puedes hacer es instalarlo.

Ejecutas el comando en la dirección donde se encuentra tu proyecto a través de la terminal.

Una vez instalado tendrás que importar la dirección de CSS y de JS porque bootstrap funciona con ambas, y esto lo puedes importar en el fichero App.js que seria el archivo donde vas a estar maquetando los componentes de tu pagina web.

Y listo con eso tendrías todo para poder empezar a usar bootstrap, si quieres usar sus componentes tan solo tienes que leer la documentación y ver como usarlos.

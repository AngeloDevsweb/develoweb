¿Cómo crear un proyecto de React Js?

Si ya conoces un poco acerca de javascript y te interesa poder trabajar con un framework o librería con la cual puedas crear una interfaz para el lado del cliente, una muy buena opción que podrías utilizar y aprender es ReactJS, esta herramienta contiene muchas bibliotecas que te permitirán poder crear una buena interfaz dependiendo de los requerimientos que tengas, así que si estas interesado en poder crear tu primer proyecto con React js te dejo esta guía para que veas como crear uno desde 0.

a instalación es bastante sencilla solo tienes que ejecutar un comando y empezara a descargar el proyecto en tu computador.

Instalación

Para empezar debemos ir a la pagina oficial de ReactJS allí copiaremos el siguiente comando: npx create-react-app my-first-app. y lo pegamos en nuestra consola de comandos si estas en Windows puedes hacerlo a través del cmd y en Linux por el terminal.

En la consola tienes que posicionarte en el directorio donde deseas instalar el proyecto y es allí donde pegaremos el comando de instalación.

Le damos a Enter y esperamos a que se descargue todo el proyecto.

Una vez descargado el proyecto tan solo tenemos que dirigirnos al directorio donde se instalo con: cd my-first-app y luego ejecutamos npm start.

REACT JS

¿Qué es React? Bien, seguramente escuchaste decir que dentro de los frameworks de javascript existe react js, pero exactamente ¿Qué es?, ¿Cómo funciona?, no te preocupes tratare de explicártelo de una forma sencilla para que entiendas un poco acerca de la funcionalidad de esta librería.

Y si, como lo leíste React js no es un framework o al menos no es considerado uno, sino más bien como una librería para javascript. Pese a la popularidad que tuvo esta biblioteca y a que la comunidad de desarrolladores le diera tanto crecimiento sigue sin ser un framework pero las funcionalidades que ofrece permite que se le considere como uno.

React es una librería que está enfocada a la creación de interfaces de usuario, con la cual podras desarrollar el frontend o el lado del cliente de una web, además de ello react es un excelente aliado para construir web SPA (Single page application). Para aquellos que no entiendan que quiere decir esto porque recién están empezando dentro del mundo del desarrollo web, te lo resumo: Las aplicaciones de una sola página o SPA por sus siglas en inglés, son páginas web que están construidas en una sola página (valga la redundancia ) y es que la características de estas webs es que cargan todos sus componentes una sola vez, es decir, la primer carga de la página es la que tarda, pero luego podrás navegar a través de todos sus componentes de una forma rápida sin tener que volver a recargar la página. Esto posibilita un uso más eficaz sobre las páginas web o bien para las aplicaciones web, que en este caso seria las más beneficiadas porque no tendrás que esperar a que el servidor responda y añadido a eso que la pagina recargue cada vez que hagas una petición.

Es por eso que esta librería se centra en ello en una SPA, con la cual usando react podrás usar componentes y estos usarlos como piezas que encajen dentro de una página web. Si lo quieres ver de otro modo es como que tengas un lienzo y luego empieces a pintar parte por parte.

¿Cómo usar React?

React te ayuda a crear interfaces de usuario interactivas de forma sencilla. Diseña vistas simples para cada estado en tu aplicación, y React se encargará de actualizar y renderizar de manera eficiente los componentes correctos cuando los datos cambien.

Crea componentes encapsulados que manejen su propio estado, y conviértelos en interfaces de usuario complejas. Ya que la lógica de los componentes está escrita en JavaScript y no en plantillas, puedes pasar datos de forma sencilla a través de tu aplicación y mantener el estado fuera del DOM.

Puedes usar React a través de una CDN pero no sería lo más recomendable o al menos yo no te lo recomiendo, porque para usar sus funcionalidades tendrías que ponerlas en cada etiqueta y eso se volvería molesto. Para ello lo que yo te recomiendo es que uses “créate-react-app” con el cual podras instalar un proyecto de react en tu escritorio, pero necesitaras también instalar antes Node js y NPM que es el manejador de paquetes de Node, con ello tendrías tu pc listo para poder instalar react.

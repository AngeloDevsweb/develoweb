VUE JS

¿Qué es Vue? de seguro te habrás preguntado alguna vez de que se trata VUE que hay detrás de esto y la verdad no es algo complejo de entender, a modo de no ser tan tecnico te explicare de manera muy sencilla que es Vue.

Vue es un framework de javascript o si quieres verlo de otro modo Vue usa como base javascript, y está construido para diseñar interfaces de usuario con el cual puedes desarrollar una página web o también una aplicación web si asi se requiere.

Vue trabaja específicamente en el lado del cliente desde donde nosotros hacemos las peticiones a un servidor que contenga una base de datos de la cual extraeremos información para luego mostrarla al usuario.

Si queremos hacer uso de Vue no será necesario instalar nada simplemente puedes trabajar con él a través de una CDN, que tan solo tienes que copiar y pegar en tu archivo html.

Vue fue creado por Evan You un ex programador de google, quien es importante nombrar ya que él fue desarrollador de angular, vue fue lanzado en el año 2014 y aunque fue inicialmente pensado para ser una librería personal, la comunidad hizo que creciera a un ritmo bastante rápido lo que hizo que se convirtiera en un framework muy utilizado.

La historia de la creación de Vue no es extensa, todo se podría resumir a que este programador ya antes mencionado, llego a sentir que angular era algo muy complicado, tedioso y eso hizo que él se planteara la idea de cómo podría crear una herramienta que recogiera lo mejor de angular y de otros frameworks y unirlo, fue entonces que de esa idea nació Vue js.

Una de las características principales de Vue es el trabajo con componentes, un componente de manera sencilla no es más que una forma de encapsular código para poder reutilizarlo las veces que se necesaria, este componente puede estar compuesto de html, css, y javascript. Y es que además de esto puedes reemplazar los componentes de una forma muy sencilla como si se trataran de piezas que pueden ser modificadas.

Otra de las principales caracterisiticas de vue es que es un framework reactivo y ¿Qué significa? pues es muy sencillo, como su nombre lo indica, es un sistema con reacción al cambio. Dependiendo de algún evento interno es posible modificar el estado de la aplicación con la cual podemos cambiar la vista de nuestra página web o aplicación.

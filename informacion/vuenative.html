¿Cómo solucionar el error de instalacion y ejecucion de vue native usando expo-cli?

Si llegaste a instalar vue native de manera correcta y sin ningún error entonces no tendrás de que preocuparte ya que al momento de ejecutar el cli de expo y capturar el QR con la app expo Go todo te funciono como debería, pero ¿y si ese no es el caso?, si a ti no te funciona al momento de ejecutar “npm start” o capturar el QR con la aplicación de expo ¿Cuál es el error?

¿Qué es lo que está fallando?, ¿Cómo lo puedo solucionar?, esas son las preguntas que seguramente te hiciste y por más que buscaste en internet, en blogs, y hasta en stack overflow no encontraste solución, a pesar de los intentos y de modificar los archivos no funciona.

Y seguro has desinstalado e instalado varias veces, pero, aun así, sigue sin solución. Bien en esta guía voy a darte una ayuda con eso, para que puedas al fin ejecutar la aplicación sin ningún error.

Primer paso. -

Para empezar lo primero que debes hacer, es revisar tu versión de node js y ¿Por qué? Pues porque actualmente a la fecha de hoy que escribo esta guía (24-03-2021), la versión más actualizada de node js es la 14.16 y npm se encuentra en la versión 7.6.3, estas son las versiones más recientes.

El problema con vue native es que sus librerías no han sido actualizadas para la versión actual de node js y npm, es por esta razón que a ti te puede estar fallando y no te ejecuta el programa o la aplicación. Si este es el problema entonces lo que tienes que hacer es instalar una versión anterior de node js.

Para esto yo te recomiendo que uses Laragon, esta es una herramienta que te será muy útil ya que podras correr node js y vue native dentro de ella. Instalarlo es muy fácil solo tienes que descargar la versión completa de Laragon y con ella vendrán instaladas diferentes tecnologías que te ayudaran en lo que necesites usar. Entre ellas vendrá instalado node js pero en una versión muy antigua que no te permitirá correr vue native, para esto tendrás que actualizarlo.

Y realizar esto es muy sencillo, simplemente tienes que ir al siguiente enlace: node js para laragon allí tienes que descargar la versión 12.13.1-win64x de node js, una vez descargado lo descomprimes y la subcarpeta la copias en la siguiente dirección: C:\laragon\bin\nodejs\node-v12.13.1

Una vez que hayas hecho esto, abres laragon y en la pestaña Menu seleccionas Node js y allí le das click en la versión 12.13.1 node js, con esto quedara seleccionado con un tick la versión que acabas de descargar.

La versión de npm es mucho más importante que la versión de node js

Ahora hare un énfasis especial en npm y es porque este es el causante principal del error que no permite que puedas ejecutar el programa expo-cli ni puedas visualizarlo en la app de expo go para empezar a desarrollar tu aplicación con vue native. Y esto es debido a que la última actualización de npm, la versión 7, ha deprecado diferentes librerías y con ellas la esencial para nosotros que es Vue-native-scripts.

Sin esta no vamos a poder ejecutar de manera correcta vue native con expo-cli, y siempre nos dara el mismo resultado, ERROR, no importa lo que intentes no te va a funcionar e incluso si intentas instalarlo como dice en su documentación oficial no te lo permitirá y solo te saldrá en la consola que no se pudo realizar la instalación y todo en error. Por esto es muy importante que no cambies la versión de npm que viene con el node js que te acabo de indicar, porque si lo actualizas a la versión 7 tendrás el mismo error nuevamente.

Segundo paso. -

Después de haberte explicado el principal error por el cual no funciona vue native, todo lo demás es muy sencillo. Solo tienes que instalar expo-cli y si no sabes cómo instalarlo te dejo aquí el link para que puedas ir a su documentación oficial: expo-cli

Copias el comando en tu terminal para instalarlo de forma global y luego empezara la descarga, una vez terminado el proceso ya tendras expo-cli para poder desplegar vue native.

Tercer paso. -

Luego de haber instalado expo-cli, ahora es el turno de instalar vue native para poder crear un proyecto con él y empezar a desarrollar con esta tecnología, al igual que en el paso anterior vamos a ir a su documentación oficial Vue native y vamos a copiar y pegar en la consola el comando para instalarlo de forma global: npm install --global vue-native-cli

Cuarto paso. -

Una vez instalado vue native tan solo nos queda crear el proyecto y eso lo haremos con el siguiente comando: vue-native init "nombreDelProyecto"

Una vez ejecutado tendremos que esperar a que se cree el proyecto que por cierto tarda un buen tiempo, luego nos aparecerá en la consola que cambiemos al directorio en donde se ha creado nuestro proyecto y allí ejecutemos el comando npm start.

Este es el cli de expo con el cual tendrás que descargarte la app expo go de la play store lo instalas y tan solo sacas captura de QR que tienes en la ventana del navegador, si hiciste bien todos los pasos verás en tu teléfono como se ejecuta el código o el texto que está escrito en el proyecto creado por vue native, donde podrás empezar a desarrollar tu propio código o realizar cambios.

Y bueno espero que esta guía te haya servido y que hayas podido entender cuál era el error que no te dejaba ejecutar vue native, en su momento yo también tuve el mismo error y no encontraba la forma de solucionarlo, hasta que después de mucho leer e intentar muchas cosas pude al fin dar con la solución.
